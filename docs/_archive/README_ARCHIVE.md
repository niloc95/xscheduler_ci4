# Documentation Archive

This folder is the canonical archive for deprecated, historical, superseded, or duplicate documentation.

## Archive Rules

- Archived files are not source-of-truth documents.
- Archived files should be prefixed with `OLD_` when moved here from active locations.
- Keep canonical active docs in their domain folders under `docs/`.
- Preserve old audits, phase reports, and implementation notes here for historical traceability only.

## Archive Groups

- `audits/` — superseded audit reports
- `completion-reports/` — completed phase and implementation reports
- `old-audits/` — older codebase-wide audit material
- `old-phases/` — historical phase planning and execution documents
- `old-summaries/` — old summary and merge documents
- `outdated-guides/` — guides replaced by newer architecture or workflows
- `old-duplicates/` — duplicate root copies archived after canonicalization
