# Security Policy

## Reporting a Vulnerability

Please **do not** disclose security issues publicly (issues, PR comments, discussions).

Report security concerns privately:
- Email: info@webschedulr.co.za

Include:
- A clear description of the issue and impact
- Steps to reproduce (proof of concept if possible)
- Affected versions/branches (if known)

## Response expectations

We aim to acknowledge reports within a few business days and will coordinate a fix and disclosure timeline with you.
